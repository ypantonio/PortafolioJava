# PortafolioJava
El programa es sobre la empresa Distribuidora Tupper Ware, tienen la finalidad de encontrar el precio de la cantidad comprada, es decir si una persona compra diez productos el programa multiplica esa cantidad con el precio al que se encuentra en la empresa, mostrando un valor unitario y la cantidad comprada, mostrando el precio total. Luego el programa tiene entendido que si se pasa de cierta cantidad o es igual le da un descuento ya sea del 5%, 15% o 25%, mostrando el descuento que se debe hacer y el precio con el descuento aplicado. Luego le da una lista del IVA que se debe pagar por la compra de los productos, las ganancias obtenidas tras la venta y el tipo de descuento que se aplic√≥ en la compra.
